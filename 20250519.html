前回の復習: 判別式

前回の授業では、2次方程式の解の性質を判別するための「判別式」について学習しました。これは、2次方程式 \(ax^2+bx+c=0\) の解の公式 \(x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}\) の中の平方根の部分 \(b^2-4ac\) にあたります。

判別式の定義と役割 📖

📖 ★★★

判別式 \(D\) は、2次方程式の解の個数や種類（実数解か虚数解か）を判別するために用いられます。

具体的には、\(D = b^2 - 4ac\) で計算されます。

判別式による解の種類の判別 📜

📜 ★★★

2次方程式 \(ax^2 + bx + c = 0\) の判別式を \(D = b^2-4ac\) とすると、解の種類は次のように判別できます。

  • \(D > 0\) のとき: 異なる2つの実数解を持つ。
  • \(D = 0\) のとき: 重解（ただ1つの実数解）を持つ。
  • \(D < 0\) のとき: 異なる2つの虚数解を持つ。（実数解を持たない）

解と係数の関係

今回は、2次方程式の解と係数の間に成り立つ重要な関係について学びます。これにより、方程式を実際に解かなくても、解の性質について多くの情報を得ることができます。

導入: 解と係数のつながり

例えば、2次方程式 \(x^2 - 5x + 6 = 0\) を考えます。これを因数分解すると \((x-2)(x-3) = 0\) となり、解は \(x = 2, 3\) です。

ここで、解の和は \(2+3 = 5\)、解の積は \(2 \times 3 = 6\) となります。これらを元の方程式の係数と比較すると、

  • 解の和 \(5\) は、\(x\) の係数 \(-5\) の符号を変えたもの (\(-(-5)\))。
  • 解の積 \(6\) は、定数項 \(6\) と一致します。

この関係は一般的に成り立ちます。

定理: 解と係数の関係 📜

📜 ★★★

2次方程式 \(ax^2 + bx + c = 0\) (ただし \(a \neq 0\)) の2つの解を \(\alpha\), \(\beta\) とすると、次の関係が成り立ちます。

  • 解の和: \(\alpha + \beta = -\frac{b}{a}\)
  • 解の積: \(\alpha \beta = \frac{c}{a}\)
⭐ ★★★

この関係式を用いることで、2次方程式を解くことなく、2つの解の和と積を係数から直接求めることができます。

例題1: 解と係数の関係の基本的な利用 ✍️ ★☆☆ ★★★

2次方程式 \(2x^2 - 3x + 5 = 0\) の2つの解を \(\alpha, \beta\) とするとき、次の式の値を求めなさい。

  1. \(\alpha + \beta\)
  2. \(\alpha\beta\)
解答・解説

解と係数の関係より、\(a=2, b=-3, c=5\)。

1. \(\alpha + \beta = -\frac{b}{a} = -\frac{-3}{2} = \frac{3}{2}\)

2. \(\alpha\beta = \frac{c}{a} = \frac{5}{2}\)

答え: 1. \(\frac{3}{2}\), 2. \(\frac{5}{2}\)

⚠️

間違いやすいポイント: 符号のミス

★★☆ 解と係数の関係を適用する際、特に解の和 \(\alpha + \beta = -\frac{b}{a}\) の計算で、マイナス符号の扱いに注意が必要です。

  • \(b\) が正の数の場合: \(\alpha+\beta = -\frac{b}{a}\) となり、結果は負の数（または \(a\) が負なら正の数）になります。
  • \(b\) が負の数の場合: \(\alpha+\beta = -\frac{-|b|}{a} = \frac{|b|}{a}\) となり、結果は正の数（または \(a\) が負なら負の数）になります。

例題1の場合、\(b=-3\) なので、\(-\frac{-3}{2}\) となり、結果は \(\frac{3}{2}\) (正の数) となります。

💡

Tips: 解と係数の関係の活用場面

★★★

  • 方程式を解かずに解の性質を知りたいとき: 解の和や積は、解そのものを求めなくても簡単に計算できます。
  • 複雑な解を持つ方程式: 解の公式を使うと計算が煩雑になる場合でも、和と積は比較的シンプルに求まります。
  • 対称式の計算: \(\alpha^2 + \beta^2\) や \(\frac{1}{\alpha} + \frac{1}{\beta}\) のように、\(\alpha\) と \(\beta\) を入れ替えても値が変わらない対称式の値を求めるのに非常に有効です。これらの式は基本対称式である \(\alpha+\beta\) と \(\alpha\beta\) を用いて表現できるためです。

根号を含む式の計算

ルート（根号）を含む式の計算にはいくつかの基本的なルールがあります。これらを正しく理解し、適用することが重要です。

根号の乗法・除法 📖

📖 ★★★

\(a > 0, b > 0\) のとき、次の計算が成り立ちます。

  • \(\sqrt{a} \times \sqrt{b} = \sqrt{ab}\) （ルート同士の掛け算は、中身同士を掛けてルートをつける）
  • \(\frac{\sqrt{a}}{\sqrt{b}} = \sqrt{\frac{a}{b}}\) （ルート同士の割り算は、中身同士を割ってルートをつける）

これらのルールは、ルートの中身をまとめて計算したり、逆にルートを分割したりする際に用いられます。

理解度チェッククイズ 1

  1. \(\sqrt{3} \times \sqrt{7}\) の計算結果として正しいものはどれですか？

    1. \(\sqrt{10}\)
    2. \(\sqrt{21}\)
    3. \(3\sqrt{7}\)
    4. \(7\sqrt{3}\)
    答えと解説

    答え: B. \(\sqrt{21}\)

    解説: \(\sqrt{a} \times \sqrt{b} = \sqrt{ab}\) の公式より、\(\sqrt{3 \times 7} = \sqrt{21}\) となります。

  2. \(\frac{\sqrt{12}}{\sqrt{3}}\) の計算結果として正しいものはどれですか？

    1. \(\sqrt{4}\)
    2. \(2\)
    3. \(\sqrt{9}\)
    4. 両方 A と B
    答えと解説

    答え: D. 両方 A と B

    解説: \(\frac{\sqrt{a}}{\sqrt{b}} = \sqrt{\frac{a}{b}}\) の公式より、\(\sqrt{\frac{12}{3}} = \sqrt{4}\)。さらに \(\sqrt{4} = 2\) です。

根号の中を簡単にする

⭐ ★★★

ルートの中の数は、できるだけ小さい自然数にするのが基本です。ルートの中に \(k^2 \times m\) のように平方数（何かの2乗の数）の因数があれば、\(k\sqrt{m}\) の形に直すことができます。

例: \(\sqrt{12} = \sqrt{4 \times 3} = \sqrt{2^2 \times 3} = 2\sqrt{3}\)

\(\sqrt{48} = \sqrt{16 \times 3} = \sqrt{4^2 \times 3} = 4\sqrt{3}\)

平方根と絶対値 📖

📖 ★★★

実数 \(a\) について、\(\sqrt{a^2}\) の計算は次のようになります。

\[ \sqrt{a^2} = |a| \]

ここで、\(|a|\) は 絶対値 を表し、次のように定義されます。

  • \(a \geq 0\) のとき: \(|a| = a\)
  • \(a < 0\) のとき: \(|a| = -a\) （例えば、\(|-3| = -(-3) = 3\)）

つまり、\(\sqrt{a^2}\) は常に0以上の値になります。

⭐ ★★★

特に \(a\) が負の数の場合、\(\sqrt{a^2} = a\) ではなく \(\sqrt{a^2} = -a\) となることに注意が必要です。例えば、\(\sqrt{(-3)^2} = \sqrt{9} = 3\) であり、これは \(-(-3)\) に等しいです。

このルールは、文字式を含むルートの計算で非常に重要になります。

例題2: 根号の計算と絶対値 ✍️

次の式を簡単にしなさい。

  1. \(\sqrt{(-5)^2}\) ★☆☆ ★★☆
  2. \(a < 0\) のとき、\(\sqrt{a^2}\) ★★☆ ★★★
  3. \(a-2 < 0\) のとき、\(\sqrt{(a-2)^2}\) ★★☆ ★★★
解答・解説

1. \(\sqrt{(-5)^2} = |-5| = 5\)

2. \(a < 0\) なので、\(|a| = -a\)。よって \(\sqrt{a^2} = -a\)。

3. \(a-2 < 0\) なので、\(|a-2| = -(a-2) = -a+2 = 2-a\)。よって \(\sqrt{(a-2)^2} = 2-a\)。

⚠️

注意点: ルートの中の2乗の外し方

★★★ \(\sqrt{A^2}\) の形が出てきたら、必ず絶対値 \(|A|\) をつけてから外す習慣をつけましょう。\(A\) の中身が正か負かを確認し、場合分けをして絶対値を外します。

  • 間違い例: \(\sqrt{(1-\sqrt{3})^2}\) を \(1-\sqrt{3}\) と計算してしまう。 \(\sqrt{3} \approx 1.732\) なので、\(1-\sqrt{3}\) は負の数です。
  • 正しい計算: \(\sqrt{(1-\sqrt{3})^2} = |1-\sqrt{3}|\)。ここで \(1-\sqrt{3} < 0\) なので、\(|1-\sqrt{3}| = -(1-\sqrt{3}) = -1+\sqrt{3} = \sqrt{3}-1\)。
  • 対策: まずは機械的に絶対値をつける。その後、絶対値の中身の符号を慎重に判断する。

分母の有理化 📖

📖 ★★★

分母の有理化（ぶんぼのゆうりか）とは、分数の分母に根号が含まれている場合に、分母と分子に同じ数を掛けることで、分母から根号をなくす変形のことです。

基本的な有理化:

\(\frac{c}{\sqrt{a}}\) の形の分数は、分母と分子に \(\sqrt{a}\) を掛けて、\(\frac{c\sqrt{a}}{a}\) とします。

例: \(\frac{1}{\sqrt{3}} = \frac{1 \times \sqrt{3}}{\sqrt{3} \times \sqrt{3}} = \frac{\sqrt{3}}{3}\)

発展的な有理化（和や差の形）:

分母が \(\sqrt{a} + \sqrt{b}\) や \(\sqrt{a} - \sqrt{b}\) の形の場合、共役な式（符号だけが異なる式）を分母と分子に掛けます。これは、乗法公式 \((x+y)(x-y) = x^2 - y^2\) を利用するものです。

  • \(\frac{c}{\sqrt{a}+\sqrt{b}} = \frac{c(\sqrt{a}-\sqrt{b})}{(\sqrt{a}+\sqrt{b})(\sqrt{a}-\sqrt{b})} = \frac{c(\sqrt{a}-\sqrt{b})}{a-b}\)
  • \(\frac{c}{\sqrt{a}-\sqrt{b}} = \frac{c(\sqrt{a}+\sqrt{b})}{(\sqrt{a}-\sqrt{b})(\sqrt{a}+\sqrt{b})} = \frac{c(\sqrt{a}+\sqrt{b})}{a-b}\)

例題3: 分母の有理化 ✍️ ★★☆ ★★★

次の式の分母を有理化しなさい。

  1. \(\frac{2}{\sqrt{6}}\)
  2. \(\frac{1}{\sqrt{5}+\sqrt{2}}\)
解答・解説

1. \(\frac{2}{\sqrt{6}}\)

分母と分子に \(\sqrt{6}\) を掛けます。

\(\frac{2 \times \sqrt{6}}{\sqrt{6} \times \sqrt{6}} = \frac{2\sqrt{6}}{6} = \frac{\sqrt{6}}{3}\)

2. \(\frac{1}{\sqrt{5}+\sqrt{2}}\)

分母と分子に、分母の共役な式である \(\sqrt{5}-\sqrt{2}\) を掛けます。

\(\frac{1 \times (\sqrt{5}-\sqrt{2})}{(\sqrt{5}+\sqrt{2})(\sqrt{5}-\sqrt{2})} = \frac{\sqrt{5}-\sqrt{2}}{(\sqrt{5})^2 - (\sqrt{2})^2} = \frac{\sqrt{5}-\sqrt{2}}{5-2} = \frac{\sqrt{5}-\sqrt{2}}{3}\)

    2. \(-(x-3)\)

  2. \(\frac{2}{3-\sqrt{7}}\) の分母を有理化した結果として正しいものはどれですか？

    1. \(3+\sqrt{7}\)
    2. \(3-\sqrt{7}\)
    3. \(\frac{3+\sqrt{7}}{2}\)
    4. \(\frac{3-\sqrt{7}}{2}\)
    答えと解説

    答え: A. \(3+\sqrt{7}\)

    解説: 分母と分子に \(3+\sqrt{7}\) を掛けます。

    \(\frac{2(3+\sqrt{7})}{(3-\sqrt{7})(3+\sqrt{7})} = \frac{2(3+\sqrt{7})}{3^2 - (\sqrt{7})^2} = \frac{2(3+\sqrt{7})}{9-7} = \frac{2(3+\sqrt{7})}{2} = 3+\sqrt{7}\)

標準的な練習問題（追加生成）

  1. 2次方程式 \(x^2 - 7x + 3 = 0\) の2つの解を \(\alpha, \beta\) とするとき、\(\alpha^3 + \beta^3\) の値を求めなさい。 ★★☆ ★★☆
    解答・解説

    解と係数の関係より、\(\alpha+\beta = 7\), \(\alpha\beta = 3\)。

    対称式の変形公式 \(\alpha^3+\beta^3 = (\alpha+\beta)^3 - 3\alpha\beta(\alpha+\beta)\) を利用します。

    \(\alpha^3+\beta^3 = (7)^3 - 3(3)(7) = 343 - 9 \times 7 = 343 - 63 = 280\)

    別解: \(\alpha^3+\beta^3 = (\alpha+\beta)(\alpha^2-\alpha\beta+\beta^2)\) を利用。

    まず \(\alpha^2+\beta^2 = (\alpha+\beta)^2 - 2\alpha\beta = 7^2 - 2(3) = 49 - 6 = 43\)。

    よって、\(\alpha^3+\beta^3 = (7)(43 - 3) = 7 \times 40 = 280\)。

  2. 次の式の分母を有理化しなさい: \(\frac{\sqrt{3}}{\sqrt{6}-\sqrt{2}}\) ★★☆ ★★☆
    解答・解説

    分母と分子に \(\sqrt{6}+\sqrt{2}\) を掛けます。

    \(\frac{\sqrt{3}(\sqrt{6}+\sqrt{2})}{(\sqrt{6}-\sqrt{2})(\sqrt{6}+\sqrt{2})} = \frac{\sqrt{18}+\sqrt{6}}{6-2} = \frac{3\sqrt{2}+\sqrt{6}}{4}\)

    （\(\sqrt{18} = \sqrt{9 \times 2} = 3\sqrt{2}\)）

  3. \(x=\frac{1}{\sqrt{3}-1}\) のとき、\(x^2+2x-1\) の値を求めなさい。 ★★★ ★★☆
    解答・解説

    まず \(x\) の分母を有理化します。

    \(x = \frac{1}{\sqrt{3}-1} = \frac{1(\sqrt{3}+1)}{(\sqrt{3}-1)(\sqrt{3}+1)} = \frac{\sqrt{3}+1}{3-1} = \frac{\sqrt{3}+1}{2}\)

    次に、与えられた式に \(x\) を代入します。

    \(x^2+2x-1 = \left(\frac{\sqrt{3}+1}{2}\right)^2 + 2\left(\frac{\sqrt{3}+1}{2}\right) - 1\)

    \(= \frac{(\sqrt{3})^2 + 2\sqrt{3} \cdot 1 + 1^2}{4} + (\sqrt{3}+1) - 1\)

    \(= \frac{3+2\sqrt{3}+1}{4} + \sqrt{3} = \frac{4+2\sqrt{3}}{4} + \sqrt{3}\)

    \(= \frac{2(2+\sqrt{3})}{4} + \sqrt{3} = \frac{2+\sqrt{3}}{2} + \frac{2\sqrt{3}}{2}\)

    \(= \frac{2+\sqrt{3}+2\sqrt{3}}{2} = \frac{2+3\sqrt{3}}{2}\)

    別解: \(x=\frac{\sqrt{3}+1}{2}\) より \(2x = \sqrt{3}+1 \Rightarrow 2x-1 = \sqrt{3}\)。

    両辺を2乗して、\((2x-1)^2 = (\sqrt{3})^2 \Rightarrow 4x^2-4x+1 = 3 \Rightarrow 4x^2-4x-2=0\)。

    両辺を2で割って、\(2x^2-2x-1=0\)。これは \(x\) が満たす関係式です。

    求める式 \(x^2+2x-1\) をこの関係式で割る（または変形する）ことを考えます。

    \(x^2+2x-1 = \frac{1}{2}(2x^2-2x-1) + 3x - \frac{1}{2}\)

    ここで \(2x^2-2x-1=0\) なので、

    \(x^2+2x-1 = 3x - \frac{1}{2} = 3\left(\frac{\sqrt{3}+1}{2}\right) - \frac{1}{2} = \frac{3\sqrt{3}+3-1}{2} = \frac{3\sqrt{3}+2}{2}\)

✏️

発展的な練習問題（追加生成）

  1. 2次方程式 \(2x^2 - (m+1)x + m-1 = 0\) が異なる2つの実数解を持つような、定数 \(m\) の値の範囲を求めなさい。 ★★★ ★★★
    解答・解説

    異なる2つの実数解を持つための条件は、判別式 \(D > 0\)。

    この方程式において、\(a=2, b=-(m+1), c=m-1\)。

    \(D = \{-(m+1)\}^2 - 4(2)(m-1)\)

    \(= (m+1)^2 - 8(m-1)\)

    \(= m^2 + 2m + 1 - 8m + 8\)

    \(= m^2 - 6m + 9\)

    \(= (m-3)^2\)

    \(D > 0\) より、\((m-3)^2 > 0\)。

    ここで、\((m-3)^2\) は \(m-3 \neq 0\) のとき常に正となります。\((m-3)^2 = 0\) となるのは \(m=3\) のときのみです。

    したがって、\((m-3)^2 > 0\) を満たす \(m\) の範囲は、\(m \neq 3\)。

    答え: \(m \neq 3\) であるすべての実数。

  2. \(a, b\) は実数とする。\(x\) についての2次方程式 \(x^2 - (a+b)x + ab - 1 = 0\) の解の1つが \(x=2\) であるとき、もう1つの解と \(a,b\) の関係式を求めなさい。 ★★★ ★★★
    解答・解説

    解の1つが \(x=2\) であるから、この値を方程式に代入すると成り立つ。

    \(2^2 - (a+b)(2) + ab - 1 = 0\)

    \(4 - 2a - 2b + ab - 1 = 0\)

    \(ab - 2a - 2b + 3 = 0 \quad \cdots (*)\)

    これが \(a,b\) の関係式の一つです。

    もう1つの解を \(\beta\) とする。解と係数の関係より、

    2つの解の和: \(2 + \beta = - \frac{-(a+b)}{1} = a+b\)

    よって、\(\beta = a+b-2\)。

    これがもう1つの解です。

    答え:

    もう1つの解: \(\beta = a+b-2\)

    \(a,b\) の関係式: \(ab - 2a - 2b + 3 = 0\) (または \( (a-2)(b-2) = 1 \) )
