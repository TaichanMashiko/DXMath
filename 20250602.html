はじめに

本日の授業では、第1章3節の最後の内容として、ワークシート7ページ目の「絶対値を含む方程式・不等式」について学習します。

1. 絶対値の定義と性質 📖 ★★★

絶対値とは、基本的に「距離」を表す記号です。特に1年生の段階では、「原点(0)からの距離」として理解しておけば問題ありません。

絶対値の定義:
数 \(a\) の絶対値を \(|a|\) と表し、次のように定義されます。

  • \(a \geq 0\) のとき、\(|a| = a\)
  • \(a < 0\) のとき、\(|a| = -a\)
例えば、\(|3| = 3\)、\(|-3| = -(-3) = 3\) となります。

2. 例題7: 絶対値を含む方程式 ✍️ ★☆☆ ★★☆

方程式 \(|x - 3| = 2\) の解を求めなさい。

解説

絶対値の定義より、「\(x-3\)」という値は、原点からの距離が2であるということになります。

数直線上で考えると、原点からの距離が2となる点は、+2 と -2 の2点です。

したがって、以下の2つの場合が考えられます。

1) \(x - 3 = 2\)
\(x = 2 + 3\)
\(x = 5\)

2) \(x - 3 = -2\)
\(x = -2 + 3\)
\(x = 1\)

⭐ 答え: \(x = 1, 5\)

3. 演習問題12 ✏️ ★☆☆ ★★☆

問12: 次の方程式を解きなさい。
(1) \(|x| = 4\)
(2) \(|x - 2| = 5\)
(3) \(|2x + 1| = 7\)

解答

(1) \(x = \pm 4\)

(2) \(x - 2 = \pm 5\)
\(x - 2 = 5 \Rightarrow x = 7\)
\(x - 2 = -5 \Rightarrow x = -3\)
よって、\(x = 7, -3\)

(3) \(2x + 1 = \pm 7\)
\(2x + 1 = 7 \Rightarrow 2x = 6 \Rightarrow x = 3\)
\(2x + 1 = -7 \Rightarrow 2x = -8 \Rightarrow x = -4\)
よって、\(x = 3, -4\)

4. 絶対値を含む不等式 📖 ★★★

絶対値を含む不等式では、絶対値の中身が正の場合と負の場合で分けて考える「場合分け」が重要になります。

5. 例題: 絶対値を含む不等式 ✍️ ★★☆ ★★★

不等式 \(|x - 3| < 2\) を解きなさい。

解説

「\(x-3\)」の絶対値が2より小さいということは、原点からの距離が2より小さい範囲にあるということです。

数直線上で考えると、原点からの距離が2の点は+2と-2です。距離が2「より小さい」範囲は、-2と+2の間になります。

したがって、\(x-3\) は -2 より大きく、かつ +2 より小さい範囲にあると考えられます。
\(-2 < x - 3 < 2\)

この形の不等式は、各辺に同じ数を足したり引いたりして解きます。今回は真ん中を \(x\) だけにしたいので、各辺に3を加えます。
\(-2 + 3 < x - 3 + 3 < 2 + 3\)
\(1 < x < 5\)

⭐ 答え: \(1 < x < 5\)

不等式 \(|x - 3| > 2\) を解きなさい。（講師が授業中に変更した例）

「\(x-3\)」の絶対値が2より大きいということは、原点からの距離が2より大きい範囲にあるということです。

数直線上で考えると、原点からの距離が2の点は+2と-2です。距離が2「より大きい」範囲は、-2より小さい部分と、+2より大きい部分になります。

したがって、以下の2つの場合が考えられます。

1) \(x - 3 < -2\)
\(x < -2 + 3\)
\(x < 1\)

2) \(x - 3 > 2\)
\(x > 2 + 3\)
\(x > 5\)

この2つの範囲は共通部分を持たないので、「または」で結びます。

⭐ 答え: \(x < 1\) または \(x > 5\)

6. 演習問題13 ✏️ ★★☆ ★★★

問13: 次の不等式を解きなさい。
(1) \(|x| < 4\)
(2) \(|x| > 4\)
(3) \(|x - 2| \leq 5\)
(4) \(|x - 2| \geq 5\)

解答

(1) \(-4 < x < 4\)

(2) \(x < -4\) または \(x > 4\)

(3) \(-5 \leq x - 2 \leq 5\)
各辺に2を加えると、\(-5 + 2 \leq x \leq 5 + 2\)
よって、\(-3 \leq x \leq 7\)

(4) \(x - 2 \leq -5\) または \(x - 2 \geq 5\)
\(x \leq -3\) または \(x \geq 7\)

7. 場合分けの重要ポイント ⭐ ★★★

絶対値を含む方程式や不等式を解く上で、最も重要な考え方の一つが「場合分け」です。

絶対値記号 \(|A|\) は、中身 \(A\) の符号によって外し方が変わります。

  • \(A \geq 0\) のとき: \(|A| = A\) (そのまま外す)
  • \(A < 0\) のとき: \(|A| = -A\) (マイナスをつけて外す)

この原則に基づいて、絶対値の中身が0以上になる場合と、0未満になる場合で分けて計算を進めます。

例: \(|x - 3|\) の場合分け
1. \(x - 3 \geq 0\) すなわち \(x \geq 3\) のとき: \(|x - 3| = x - 3\)
2. \(x - 3 < 0\) すなわち \(x < 3\) のとき: \(|x - 3| = -(x - 3) = -x + 3\)

⚠️ 場合分けした後は、それぞれの条件の下で得られた解が、その条件を満たしているかを確認する必要があります。条件を満たさない解は除外します。最後に、各場合で得られた解を合わせます。

理解度チェッククイズ ❓ ★★☆

絶対値を含む方程式 \(|2x - 1| = 5\) を解く場合、絶対値の中身 \(2x-1\) で場合分けすると、どのような2つの場合を考えますか？

  1. \(2x-1 \geq 0\) の場合と \(2x-1 < 0\) の場合
  2. \(x \geq 0\) の場合と \(x < 0\) の場合
  3. \(2x-1 = 5\) の場合と \(2x-1 = -5\) の場合
解答と解説

解答: a. \(2x-1 \geq 0\) の場合と \(2x-1 < 0\) の場合

解説: 絶対値記号を外すための場合分けは、絶対値の中身の式が0以上か0未満かで分けます。この問題では、\(2x-1\) がその対象です。cは場合分けではなく、絶対値の定義から直接導かれる方程式です。

⚠️ 間違いやすいポイント ★★☆

  • 絶対値の外し方: 絶対値の中身が負の場合、符号を変えて（マイナスを付けて）外すことを忘れがちです。
  • 不等号の向き（場合分け後）: 場合分けして不等式を解く際に、基本的な不等式の操作（負の数で割るなど）で不等号の向きが変わるルールは依然として適用されます。
  • 解の吟味忘れ: 場合分けの条件と、その条件下で得られた解が矛盾していないか（条件を満たしているか）の確認（吟味）を怠ると、不適切な解を含めてしまうことがあります。
  • \(|A| > c\) ( \(c>0\) ) の解き方: \(|A| > c\) の解は \(A < -c\) または \(A > c\) となります。「かつ」ではなく「または」で結ばれる2つの範囲になる点に注意が必要です。数直線で考えると、原点からの距離が \(c\) より大きい範囲、つまり \(-c\) より左側と \(c\) より右側になります。
  • \(|A| < c\) ( \(c>0\) ) の解き方: \(|A| < c\) の解は \(-c < A < c\) となります。これは1つの連続した範囲です。数直線で考えると、原点からの距離が \(c\) より小さい範囲、つまり \(-c\) と \(c\) の間になります。

✏️ 標準的な練習問題 ★★☆

問題 1 ★★☆

方程式 \(|3x + 2| = x + 6\) を解きなさい。

解答と解説

場合分けを行います。

[1] \(3x + 2 \geq 0\) すなわち \(x \geq -\frac{2}{3}\) のとき
絶対値はそのまま外せるので、
\(3x + 2 = x + 6\)
\(2x = 4\)
\(x = 2\)
これは \(x \geq -\frac{2}{3}\) を満たします。

[2] \(3x + 2 < 0\) すなわち \(x < -\frac{2}{3}\) のとき
絶対値はマイナスをつけて外すので、
\(-(3x + 2) = x + 6\)
\(-3x - 2 = x + 6\)
\(-4x = 8\)
\(x = -2\)
これは \(x < -\frac{2}{3}\) を満たします。

したがって、求める解は \(x = 2, -2\)。

⭐ 答え: \(x = 2, -2\)

問題 2 ★★☆

不等式 \(|x - 4| \leq 2x + 1\) を解きなさい。

解答と解説

場合分けを行います。

[1] \(x - 4 \geq 0\) すなわち \(x \geq 4\) のとき
絶対値はそのまま外せるので、
\(x - 4 \leq 2x + 1\)
\(-x \leq 5\)
\(x \geq -5\)
条件 \(x \geq 4\) との共通範囲は \(x \geq 4\)。

[2] \(x - 4 < 0\) すなわち \(x < 4\) のとき
絶対値はマイナスをつけて外すので、
\(-(x - 4) \leq 2x + 1\)
\(-x + 4 \leq 2x + 1\)
\(-3x \leq -3\)
\(x \geq 1\)
条件 \(x < 4\) との共通範囲は \(1 \leq x < 4\)。

最後に、[1]と[2]で得られた解を合わせます（和集合）。
\(x \geq 4\) または \(1 \leq x < 4\)
これを数直線で考えると、\(x \geq 1\) となります。

⭐ 答え: \(x \geq 1\)

✏️ 発展的な練習問題 ★★★

問題 1 ★★★

不等式 \(|x| + |x - 2| < 4\) を解きなさい。

解答と解説

絶対値の中身が0になる \(x\) の値は、\(x=0\) と \(x-2=0 \Rightarrow x=2\) です。これにより、数直線を3つの区間に分けて場合分けします。

[1] \(x < 0\) のとき
このとき、\(x < 0\) かつ \(x - 2 < 0\) なので、
\(|x| = -x\)
\(|x - 2| = -(x - 2) = -x + 2\)
よって、不等式は、
\(-x + (-x + 2) < 4\)
\(-2x + 2 < 4\)
\(-2x < 2\)
\(x > -1\)
条件 \(x < 0\) との共通範囲は \(-1 < x < 0\)。

[2] \(0 \leq x < 2\) のとき
このとき、\(x \geq 0\) かつ \(x - 2 < 0\) なので、
\(|x| = x\)
\(|x - 2| = -(x - 2) = -x + 2\)
よって、不等式は、
\(x + (-x + 2) < 4\)
\(2 < 4\)
これは常に成り立ちます。したがって、この区間 \(0 \leq x < 2\) はすべて解となります。

[3] \(x \geq 2\) のとき
このとき、\(x > 0\) かつ \(x - 2 \geq 0\) なので、
\(|x| = x\)
\(|x - 2| = x - 2\)
よって、不等式は、
\(x + (x - 2) < 4\)
\(2x - 2 < 4\)
\(2x < 6\)
\(x < 3\)
条件 \(x \geq 2\) との共通範囲は \(2 \leq x < 3\)。

最後に、[1], [2], [3]で得られた解を合わせます。
\(-1 < x < 0\), \(0 \leq x < 2\), \(2 \leq x < 3\)
これらを数直線上で合わせると、\(-1 < x < 3\) となります。

⭐ 答え: \(-1 < x < 3\)

🔖 授業の要点まとめ ★★★

  • 絶対値 \(|A|\) は、原点からの距離を表し、\(A \geq 0\) のとき \(A\)、\(A < 0\) のとき \(-A\) となります。
  • 絶対値を含む方程式 \(|A| = c\) (\(c \geq 0\)) の解は \(A = c\) または \(A = -c\) です。
  • 絶対値を含む不等式 \(|A| < c\) (\(c > 0\)) の解は \(-c < A < c\) です。
  • 絶対値を含む不等式 \(|A| > c\) (\(c > 0\)) の解は \(A < -c\) または \(A > c\) です。
  • 絶対値の記号が複数ある場合や、絶対値の外側にも変数がある場合は、絶対値の中身の符号によって場合分けをして解きます。
  • 場合分けした際は、各場合で得られた解が、その場合分けの条件を満たしているか（解の吟味）を確認することが不可欠です。
