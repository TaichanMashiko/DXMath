例6: 基本的な連立不等式 ★☆☆ ★★☆

以下の連立不等式を解いてみましょう。

\[ \begin{cases} 2x - 3 \leq 4x - 7 & \cdots ① \\ 8x + 7 \leq 5x - 2 & \cdots ② \end{cases} \]

解説（問題文に基づく解法と授業中の解説について）

まず、提示された問題文に基づいて各不等式を個別に解きます。

不等式①の解法 (問題文に基づく): \(2x - 3 \leq 4x - 7\)

xの項を右辺に、定数項を左辺に移項します。
\(-3 + 7 \leq 4x - 2x\)
\(4 \leq 2x\)
両辺を2で割ります。
\(2 \leq x\)
すなわち、\(x \geq 2\) です。

不等式②の解法 (問題文に基づく): \(8x + 7 \leq 5x - 2\)

xの項を左辺に、定数項を右辺に移項します。
\(8x - 5x \leq -2 - 7\)
\(3x \leq -9\)
両辺を3で割ります。
\(x \leq -3\) です。

次に、得られた解 \(x \geq 2\) と \(x \leq -3\) の共通範囲を数直線上で考えます。

\(x \geq 2\) は数直線上で2を含み右側の範囲、\(x \leq -3\) は-3を含み左側の範囲です。
これらの範囲に共通部分はないため、この連立不等式の解は「解なし」となります。

⭐ 答え（問題文に基づく）: 解なし

➕ 授業中の解説について

授業中の解説では、講師は不等式①の解を \(x > 2\)（問題文の \(2x-3 \leq 4x-7\) からは \(x \geq 2\) が導かれますが、解説では2を含まない白丸で説明していました）、不等式②の解を \(x \geq 3\)（問題文の \(8x+7 \leq 5x-2\) からは \(x \leq -3\) が導かれます）として扱っていました。

講師が解説で目指していたのは、これらの共通範囲 \(x \geq 3\) を求めることだったと推測されます。これは、講師が異なる問題設定（例えば、不等号の向きや等号の有無が異なるもの）を念頭に置いていたか、あるいは解説の途中で条件が変わったためと考えられます。以下に、講師が授業中に説明した \(x > 2\) と \(x \geq 3\) の組み合わせで共通範囲を求める場合の数直線を示します。

この場合の共通範囲は \(x \geq 3\) となります。授業では、このような共通範囲が存在するケースを念頭に、数直線の読み取り方の練習が行われたものと解釈できます。

⭐ 授業中の解説で目指した解（推測）: \(x \geq 3\)

理解度チェッククイズ ❓ ★☆☆

  1. 連立不等式を解く際の最初のステップは何ですか？
    1. 共通範囲を数直線で探す
    2. それぞれの不等式を個別に解く
    3. 不等式のグラフを描く
    解答と解説

    解答: b. それぞれの不等式を個別に解く

    解説: 連立不等式では、まず各不等式が示すxの範囲を求め、その後に共通範囲を探します。

例題3: A < B < C の形の不等式 ★★☆ ★★★

不等式 \(3x - 8 < x + 4 < -2x + 7\) を解きなさい。

解説

この形の不等式は、\(A < B\) かつ \(B < C\) の形の連立不等式として考えます。

すなわち、
\(3x - 8 < x + 4 \quad \cdots ①\)
\(x + 4 < -2x + 7 \quad \cdots ②\)
の2つの不等式を同時に満たす範囲を求めます。

不等式①の解法: \(3x - 8 < x + 4\)

\(3x - x < 4 + 8\)
\(2x < 12\)
\(x < 6\)

不等式②の解法: \(x + 4 < -2x + 7\)

\(x + 2x < 7 - 4\)
\(3x < 3\)
\(x < 1\)

得られた解 \(x < 6\) と \(x < 1\) の共通範囲を数直線上で考えます。

\(x < 6\) は6を含まず左側の範囲、\(x < 1\) は1を含まず左側の範囲です。
これらの共通範囲は \(x < 1\) となります。

⭐ 答え: \(x < 1\)

理解度チェッククイズ ❓ ★★☆

  1. 不等式 \(2 < x+1 \leq 5\) を解く場合、どのように2つの不等式に分解しますか？
    1. \(2 < x+1\) と \(2 \leq 5\)
    2. \(2 < x+1\) と \(x+1 \leq 5\)
    3. \(x+1 < 2\) と \(x+1 \leq 5\)
    解答と解説

    解答: b. \(2 < x+1\) と \(x+1 \leq 5\)

    解説: \(A < B \leq C\) の形の不等式は、\(A < B\) かつ \(B \leq C\) として分解します。

⚠️ 間違いやすいポイント ★★☆

  • 不等号の向き: 不等式の両辺に負の数を掛けたり割ったりするときは、不等号の向きが逆転することを見落としがちです。常に意識しましょう。
  • \(A < B < C\) の分解ミス: \(A < B < C\) という形の不等式を解く際、\(A < B\) と \(A < C\) のように誤って分解してしまうことがあります。正しくは、\(A < B\) かつ \(B < C\) と分解します。
  • 共通範囲の誤解: 数直線上で共通範囲を求める際に、「かつ」と「または」を混同しやすいです。連立不等式の解は、全ての不等式を同時に満たす範囲（積集合）なので、「かつ」の範囲を正確に読み取る必要があります。
  • 等号の有無: 不等号に等号（\(\leq, \geq\)）が含まれるか否かで、解の端点を含むかどうかが変わります。数直線上では、含む場合は黒丸（●）、含まない場合は白丸（○）で区別し、最終的な解にも正しく反映させる必要があります。

🧭 連立不等式の解法ステップガイド ★★★

連立不等式や \(A < B < C\) 型の不等式を解く際の基本的なステップは以下の通りです。

  • 1. 不等式の分解（必要な場合）:
    • 通常の連立不等式: そのまま各不等式を扱います。
    • \(A < B < C\) 型の不等式: \(A < B\) と \(B < C\) の2つの不等式に分解します。
  • 2. 各不等式の単純化:
    • 移項や両辺への加減乗除を行い、それぞれの不等式を \(x > a\), \(x \leq b\) のような最も単純な形に変形します。
    • 注意: 負の数で両辺を掛けたり割ったりする際は、不等号の向きを逆にすることを忘れないでください。
  • 3. 数直線による図示:
    • 得られた各不等式の解を、一本の数直線上に図示します。
    • 等号を含む場合は黒丸（●）、含まない場合は白丸（○）で端点を明確にします。
    • 各解が示す範囲を矢印で示します。
  • 4. 共通範囲の特定:
    • 数直線上で、全ての不等式の解が重なっている部分（共通範囲）を見つけます。
    • この共通範囲が、連立不等式の最終的な解となります。
    • 共通範囲が存在しない場合は「解なし」となります。
  • 5. 解の記述:
    • 特定した共通範囲を、不等式を用いて正確に記述します。例: \(a < x \leq b\), \(x > c\), 解なし。

✏️ 標準的な練習問題 ★★☆

問題 1 ★☆☆

次の連立不等式を解きなさい。

\[ \begin{cases} 3x + 5 > -4 \\ 2x - 1 \leq x + 3 \end{cases} \]

解答と解説

① \(3x + 5 > -4\)
\(3x > -4 - 5\)
\(3x > -9\)
\(x > -3\)

② \(2x - 1 \leq x + 3\)
\(2x - x \leq 3 + 1\)
\(x \leq 4\)

共通範囲は \(-3 < x \leq 4\)。

答え: \(-3 < x \leq 4\)

問題 2 ★★☆

次の不等式を解きなさい。

\[-x < 2x + 6 \leq 15\]

解答と解説

この不等式は、次の2つの不等式に分解して考えます。
① \(-x < 2x + 6\)
② \(2x + 6 \leq 15\)

① \(-x < 2x + 6\)
\(-x - 2x < 6\)
\(-3x < 6\)
両辺を-3で割ると不等号の向きが変わります。
\(x > -2\)

② \(2x + 6 \leq 15\)
\(2x \leq 15 - 6\)
\(2x \leq 9\)
\(x \leq \frac{9}{2}\)

共通範囲は \(-2 < x \leq \frac{9}{2}\)。

答え: \(-2 < x \leq \frac{9}{2}\)

✏️ 発展的な練習問題 ★★★

問題 1: 文章題 ★★★

あるクラスで、生徒にクッキーを配ります。1人に5個ずつ配ると12個余り、1人に7個ずつ配ると4個以上不足します。このクラスの生徒の人数として考えられる最小の人数と最大の人数を求めなさい。

解答と解説

生徒の人数を \(x\) 人とします。(ただし \(x\) は正の整数)

条件1: 「1人に5個ずつ配ると12個余る」
クッキーの総数は \(5x + 12\) 個と表せます。

条件2: 「1人に7個ずつ配ると4個以上不足する」
これは、もし \(x\) 人全員に7個ずつ配ろうとした場合、用意されているクッキーの個数 \(5x+12\) は、\(7x\) 個に対して4個以上足りない状態です。 つまり、必要な個数 \(7x\) から実際の個数 \(5x+12\) を引いた差（不足分）が4以上である、と考えることができます。
\(7x - (5x + 12) \geq 4\)
\(7x - 5x - 12 \geq 4\)
\(2x - 12 \geq 4\)
\(2x \geq 16\)
\(x \geq 8 \quad \cdots (A)\)

また、「不足する」という状況から、クッキーの総数 \(5x+12\) は、\(x\) 人全員に7個ずつ配るには足りない、つまり \(7x\) 個よりは少ないと言えます。
\(5x + 12 < 7x\)
\(12 < 2x\)
\(6 < x \quad \cdots (B)\)

さらに、条件2を別の角度から考えると、 \(x-1\) 人には7個ずつ配ることができ、最後の1人(\(x\)人目)には7個配れない（4個以上不足するため、配れるのは最大でも \(7-4=3\) 個）と解釈できます。 クッキーの総数 \(5x+12\) は、\(x-1\) 人に7個ずつ配った数 \(7(x-1)\) よりは多く（または等しく、0個配る場合）、\(7(x-1)\) に \(x\)人目に配れる最大個数3を加えたもの以下になります。
\(7(x-1) \leq 5x+12 \leq 7(x-1) + 3\)
この連立不等式を解きます。
左側: \(7x - 7 \leq 5x + 12 \Rightarrow 2x \leq 19 \Rightarrow x \leq 9.5 \quad \cdots (C)\)
右側: \(5x + 12 \leq 7x - 7 + 3 \Rightarrow 5x + 12 \leq 7x - 4 \Rightarrow 16 \leq 2x \Rightarrow x \geq 8 \quad \cdots (D)\)

条件 (A), (B), (C), (D) をすべて満たす整数 \(x\) を考えます。
(A) \(x \geq 8\)
(B) \(x > 6\)
(C) \(x \leq 9.5\)
(D) \(x \geq 8\)
これらの共通範囲は \(8 \leq x \leq 9.5\) です。
\(x\) は整数なので、考えられる \(x\) は 8 または 9 です。

したがって、生徒の人数として考えられる最小の人数は8人、最大の人数は9人です。

答え: 最小8人、最大9人

💡 Tips/補足説明 ★★☆

  • 数直線の活用: 連立不等式の解を視覚的に理解するために、数直線は非常に有効なツールです。特に範囲が複雑な場合や、「解なし」となる可能性を判断する際に役立ちます。等号の有無を白丸・黒丸で正確に表現しましょう。
  • 移項の基本: 不等式でも方程式と同様に、項を移項する際は符号を変えます。この基本操作を間違えないように注意しましょう。
  • A < B < C の変形: この形の不等式は、必ず \(A < B\) と \(B < C\) の2つの部分に分けて考えます。\(A < C\) を単独で使わないように注意してください。
  • 検算の習慣: もし時間に余裕があれば、得られた解の範囲から適当な値を元の不等式に代入して、成り立っているか確認する（検算する）習慣をつけると、ミスを減らせます。例えば、\(x < 1\) が解ならば、\(x=0\) を代入して確認するなど。
  • 文章題の立式: 文章題から不等式を立てる際は、問題文の「以上」「以下」「より大きい（超過）」「より小さい（未満）」といった言葉を正確に不等号に置き換えることが重要です。

🔖 授業の要点まとめ ★★★

  • 連立不等式を解くには、まずそれぞれの不等式を個別に解き、その後に共通の範囲（解）を求めます。
  • \(A < B < C\) の形の不等式は、\(A < B\) かつ \(B < C\) という連立不等式として扱います。
  • 数直線を利用すると、複数の不等式の解の共通範囲を視覚的に把握しやすくなります。
  • 不等式の両辺に負の数を掛けたり割ったりする際は、不等号の向きが逆になることに注意が必要です。
  • 最終的な解は、全ての条件を満たす範囲であり、「解なし」となる場合もあります。
