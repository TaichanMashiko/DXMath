1. 関数の定義と表記 📖 ★★★

関数とは何か？

関数とは、簡単に言うと「関係する数」のことです。ある値（例えば \(x\)）が決まると、それに対応して別の値（例えば \(y\)）がただ一つ決まる関係を指します。

関数: 変数 \(x\) の値を定めると、それに対応して変数 \(y\) の値がただ一つ定まるとき、\(y\) は \(x\) の関数であるといいます。

例：

  • 小学校で学習した比例: \(y = 2x\) （\(x\) が決まれば \(y\) がただ一つ決まる）
  • 中学校で学習した一次関数: \(y = 2x + 1\)
  • 中学校で学習した二次関数: \(y = x^2\)

⚠️ 関数ではない例: \(y^2 = x\)。例えば \(x=4\) のとき、\(y^2=4\) より \(y = \pm 2\) となり、\(y\) の値が2つ定まってしまうため、これは \(y\) が \(x\) の関数であるとは言えません（ただし、\(x\) が \(y\) の関数であるとは言えます）。

関数の表記方法: y=f(x) ★★★

関数は、しばしば \(y = f(x)\) のように表記されます。ここで、\(f\) は英語の function（関数）の頭文字です。これは「\(y\) は \(x\) の関数である」という意味を表します。

\(f(x)\) は「エフ・エックス」と読みます。\(f(x)\) の \(x\) の部分には具体的な数値が入ることがあり、例えば \(f(1)\) は「関数 \(f(x)\) の \(x\) に1を代入したときの \(y\) の値」を意味します。

✍️ 例: \(f(x) = 5x - 2\) のとき

  • \(f(1)\) は \(x=1\) を代入して、\(5 \times 1 - 2 = 3\)
  • \(f(-2)\) は \(x=-2\) を代入して、\(5 \times (-2) - 2 = -10 - 2 = -12\)
  • \(f(a)\) は \(x=a\) を代入して、\(5a - 2\)

2. 定義域と値域 📖 ★★★

定義域について

定義域（ていぎいき）: 関数 \(y=f(x)\) において、変数 \(x\) がとりうる値の範囲のこと。

定義域は、問題文で指定される場合と、関数の形から暗黙的に決まる場合があります。

例: \(y = \frac{1}{x}\) の場合、分母が0になってはいけないので、\(x \neq 0\) が暗黙の定義域となります。

値域について

値域（ちいき）: 関数 \(y=f(x)\) において、定義域の \(x\) の値に対応して \(y\) がとりうる値の範囲のこと。

値域は、定義域と関数の形によって決まります。

グラフを用いた視覚的理解

関数 \(y = x^2\) のグラフを例に、定義域と値域を考えてみましょう。

  • もし定義域が指定されていなければ、実数全体と考えられます。このとき、\(y\) は \(y \geq 0\) の範囲の値をとり、これが値域となります。
  • もし定義域が \(0 \leq x \leq 2\) と指定された場合、\(x\) は0から2までの値をとります。このとき、\(y\) は \(y = 0^2 = 0\) から \(y = 2^2 = 4\) までの値をとり、値域は \(0 \leq y \leq 4\) となります。

3. 例題演習 ✍️

例題1: 一次関数の定義域と値域 ★☆☆ ★★☆

関数 \(y = -2x + 3\) について、定義域が \(-1 \leq x \leq 2\) のときの値域を求めなさい。

解答と解説

この関数は一次関数であり、傾きが-2（負）なので、グラフは右下がりの直線になります。

定義域の端点における \(y\) の値を求めます。

  • \(x = -1\) のとき: \(y = -2(-1) + 3 = 2 + 3 = 5\)
  • \(x = 2\) のとき: \(y = -2(2) + 3 = -4 + 3 = -1\)

グラフが右下がりなので、\(x\) が増加すると \(y\) は減少します。したがって、

  • 定義域の最小値 \(x=-1\) のときに \(y\) は最大値 \(5\) をとる。
  • 定義域の最大値 \(x=2\) のときに \(y\) は最小値 \(-1\) をとる。

よって、値域は \(-1 \leq y \leq 5\) となります。

⭐ 答え: \(-1 \leq y \leq 5\)

理解度チェッククイズ ❓ ★☆☆

  1. 関数 \(y = 3x - 1\) の定義域が \(0 \leq x \leq 3\) のとき、値域はどうなりますか？
    1. \(-1 \leq y \leq 8\)
    2. \(0 \leq y \leq 3\)
    3. \(2 \leq y \leq 8\)
    解答と解説

    解答: a. \(-1 \leq y \leq 8\)

    解説: \(x=0\) のとき \(y = 3(0) - 1 = -1\)。\(x=3\) のとき \(y = 3(3) - 1 = 9 - 1 = 8\)。傾きが正なので、値域は \(-1 \leq y \leq 8\) となります。

⚠️ 間違いやすいポイント ★★☆

  • 定義域と値域の混同: \(x\) の範囲が定義域、\(y\) の範囲が値域です。どちらを問われているのかを正確に把握しましょう。
  • グラフの傾きと値域: 一次関数の場合、グラフが右上がり（傾きが正）か右下がり（傾きが負）かで、定義域の端点が値域の最大・最小にどう対応するかが変わります。二次関数の場合は、頂点の位置が重要になります。
  • 関数の表記 \(f(x)\) の誤解: \(f(x)\) は \(f \times x\) ではなく、\(x\) の関数 \(f\) であることを示します。\(f(3)\) は \(x=3\) を関数に代入したときの \(y\) の値を意味します。
  • 「ただ一つ決まる」の理解: 関数の定義において「\(y\) の値がただ一つ決まる」という点が重要です。\(x\) の値に対して複数の \(y\) が対応する場合は関数ではありません。

✏️ 標準的な練習問題

問題 1 ★☆☆ ★★☆

関数 \(f(x) = x^2 - 3x + 5\) について、次の値を求めなさい。

  1. \(f(0)\)
  2. \(f(2)\)
  3. \(f(-1)\)
解答と解説
  1. \(f(0) = 0^2 - 3(0) + 5 = 0 - 0 + 5 = 5\)
  2. \(f(2) = 2^2 - 3(2) + 5 = 4 - 6 + 5 = 3\)
  3. \(f(-1) = (-1)^2 - 3(-1) + 5 = 1 + 3 + 5 = 9\)

答え: 1. 5, 2. 3, 3. 9

問題 2 ★★☆ ★★☆

関数 \(y = \frac{1}{2}x + 1\) について、定義域が \(-2 \leq x < 4\) のときの値域を求めなさい。

解答と解説

この関数は一次関数で、傾きが \(\frac{1}{2}\)（正）なので、グラフは右上がりの直線です。

定義域の端点における \(y\) の値を求めます。

  • \(x = -2\) のとき: \(y = \frac{1}{2}(-2) + 1 = -1 + 1 = 0\)
  • \(x = 4\) のとき: \(y = \frac{1}{2}(4) + 1 = 2 + 1 = 3\)

グラフが右上がりなので、\(x\) が増加すると \(y\) も増加します。 定義域の左端 \(x=-2\) は含まれるため、\(y=0\) は値域に含まれます。 定義域の右端 \(x=4\) は含まれないため、\(y=3\) は値域に含まれません。

よって、値域は \(0 \leq y < 3\) となります。

⭐ 答え: \(0 \leq y < 3\)

🔖 授業の要点まとめ ★★★

  • 関数: 変数 \(x\) の値を定めると、それに対応して変数 \(y\) の値がただ一つ定まるとき、\(y\) は \(x\) の関数である。表記は \(y=f(x)\) など。
  • 定義域: 関数 \(y=f(x)\) において、変数 \(x\) がとりうる値の範囲。
  • 値域: 関数 \(y=f(x)\) において、定義域の \(x\) の値に対応して \(y\) がとりうる値の範囲。
  • 定義域が指定されている場合、関数のグラフ（直線の傾きや放物線の頂点など）を考慮して値域を求める。
