絶対値の復習 📖

★★★

まず、絶対値の基本的な考え方を確認しましょう。

定義：原点からの距離

★★★

実数 \(a\) の絶対値 \(|a|\) は、数直線上で原点 \(0\) から点 \(a\) までの距離を表します。

距離なので、絶対値は常に \(0\) 以上になります。

  • 例: \(|2| = 2\), \(|-3| = 3\), \(|0| = 0\)
⭐

絶対値は「距離」であるという定義をしっかり理解することが重要です。これが方程式や不等式を解く上での基本となります。

絶対値の外し方

★★★

絶対値記号を外すときは、中身の数が正か負（または0）かで場合分けします。

  • \( a \ge 0 \) のとき： \(|a| = a\) （そのまま外す）
  • \( a < 0 \) のとき： \(|a| = -a\) （マイナスをつけて外す）

例:

  • \(|5| = 5\) (中身が正なのでそのまま)
  • \(|-5| = -(-5) = 5\) (中身が負なのでマイナスをつける)
  • \(|0| = 0\) (中身が0なのでそのまま)
⚠️

\(a < 0\) のときに \(|a| = -a\) となるのは、「\(a\) 自体が負の数なので、それにマイナスをつけることで正の数（距離）にする」という意味です。\(-a\) という表記自体が負の数を意味するわけではありません。

基本的な性質（再確認）

★★☆
➕
  • \(|a| \ge 0\)
  • \(|a| = |-a|\)
  • \(|ab| = |a||b|\)
  • \(|\frac{a}{b}| = \frac{|a|}{|b|}\) (ただし \(b \neq 0\))
  • \(|a|^2 = a^2\)

絶対値を含む方程式 ✍️

★★★

絶対値の定義（原点からの距離）に基づいて、絶対値を含む方程式を解いてみましょう。

基本形：|x| = a (a > 0)

★★★

方程式 \(|x| = a\) （ただし \(a > 0\)）は、「原点からの距離が \(a\) である点 \(x\) を求めなさい」という意味です。

数直線を考えると、原点から距離が \(a\) である点は \(+a\) と \(-a\) の2つあります。

よって、\(|x| = a\) の解は \(x = \pm a\) となります。

⚠️ \(a < 0\) の場合、\(|x| = a\) を満たす実数 \(x\) は存在しません（距離は負にならないため）。\(a=0\) の場合は \(|x|=0\) となり、解は \(x=0\) のみです。

例題：|x| = 3

★☆☆

原点からの距離が3である点を求めます。

解答

\(x = \pm 3\)

応用形：|x - p| = a

★★★

\(|x - p|\) は、数直線上での「点 \(x\) と点 \(p\) の間の距離」と解釈できます。（これは \(|p - x|\) とも同じです）

したがって、方程式 \(|x - p| = a\) （ただし \(a > 0\)）は、「点 \(p\) からの距離が \(a\) である点 \(x\) を求めなさい」という意味になります。

点 \(p\) から距離 \(a\) の点は、\(p+a\) と \(p-a\) の2つあります。

よって、\(|x - p| = a\) の解は \(x - p = \pm a\)、つまり \(x = p \pm a\) となります。

例題：|x - 2| = 5

★☆☆

点 \(2\) からの距離が \(5\) である点 \(x\) を求めます。

解答

考え方1（距離で考える）：点 \(2\) から距離 \(5\) の点は \(2+5=7\) と \(2-5=-3\) です。よって \(x = 7, -3\)

考え方2（定義に従って外す）：

\( x - 2 = \pm 5 \)

場合1: \( x - 2 = 5 \implies x = 7 \)

場合2: \( x - 2 = -5 \implies x = -3 \)

よって \(x = 7, -3\)

✏️

練習問題（方程式）

次の方程式を解きなさい。

  1. \( |x| = 5 \) ★☆☆ ★★☆
    解答

    \(x = \pm 5\)

  2. \( |x + 3| = 1 \) ★☆☆ ★★☆
    解答

    \(x + 3 = \pm 1\)。\(x+3=1\) より \(x=-2\)。\(x+3=-1\) より \(x=-4\)。よって \(x = -2, -4\)

  3. \( |2x - 1| = 7 \) ★★☆ ★★★
    解答

    \(2x - 1 = \pm 7\)。\(2x-1=7\) より \(2x=8, x=4\)。\(2x-1=-7\) より \(2x=-6, x=-3\)。よって \(x = 4, -3\)

❓

理解度チェッククイズ1

  1. 方程式 \( |-2| = x \) の解は何ですか？
    1. \( x = 2 \)
    2. \( x = -2 \)
    3. \( x = \pm 2 \)
    解答と解説

    正解は a. \( x = 2 \) です。

    まず左辺の絶対値を計算します。\( |-2| = 2 \) です。よって方程式は \( 2 = x \) となり、解は \(x=2\) です。

  2. 方程式 \( |x - 5| = 0 \) の解は何ですか？
    1. 解なし
    2. \( x = 0 \)
    3. \( x = 5 \)
    解答と解説

    正解は c. \( x = 5 \) です。

    絶対値が \(0\) になるのは、中身が \(0\) のときだけです。よって \( x - 5 = 0 \) を解いて \(x=5\) となります。「点 \(5\) からの距離が \(0\) である点 \(x\) は？」と考えても \(x=5\) です。

絶対値を含む不等式 ✍️

★★★

次に、絶対値を含む不等式の解き方を学びます。これも「原点からの距離」という定義に基づいて考えます。

基本形1：|x| < a (a > 0)

★★★

不等式 \(|x| < a\) （ただし \(a > 0\)）は、「原点からの距離が \(a\) より小さい範囲 \(x\) を求めなさい」という意味です。

数直線上で考えると、原点からの距離が \(a\) より小さいのは、\(-a\) と \(a\) の間の範囲になります。

よって、\(|x| < a\) の解は \( -a < x < a \) となります。

⚠️ \(a \le 0\) の場合、\(|x| < a\) を満たす実数 \(x\) は存在しません（距離は負や0より小さくはなれないため）。

例題：|x| < 3

★☆☆

原点からの距離が3より小さい範囲を求めます。

解答

\(-3 < x < 3\)

基本形2：|x| > a (a > 0)

★★★

不等式 \(|x| > a\) （ただし \(a > 0\)）は、「原点からの距離が \(a\) より大きい範囲 \(x\) を求めなさい」という意味です。

数直線上で考えると、原点からの距離が \(a\) より大きいのは、\(a\) より大きい範囲と、\(-a\) より小さい範囲になります。

よって、\(|x| > a\) の解は \( x < -a \) または \( x > a \) となります。

➕ \(a \le 0\) の場合、\(|x| > a\) の解はすべての実数となります（絶対値は常に0以上なので）。

例題：|x| > 3

★☆☆

原点からの距離が3より大きい範囲を求めます。

解答

\( x < -3, x > 3 \)

応用形：|x - p| < a, |x - p| > a

★★★

方程式と同様に、\(|x - p|\) を「点 \(p\) からの距離」と考えると、応用的な不等式も解けます。

  • \(|x - p| < a\) (a > 0) : 点 \(p\) からの距離が \(a\) より小さい範囲。

    解: \( -a < x - p < a \implies p - a < x < p + a \)

  • \(|x - p| > a\) (a > 0) : 点 \(p\) からの距離が \(a\) より大きい範囲。

    解: \( x - p < -a \) または \( x - p > a \)

    \( \implies x < p - a \) または \( x > p + a \)

例題：|x - 2| < 5

★☆☆

点 \(2\) からの距離が \(5\) より小さい範囲 \(x\) を求めます。

解答

考え方1（距離で考える）：点 \(2\) から距離 \(5\) の点は \(7\) と \(-3\) です。距離が \(5\) より小さいのは、この2点の間の範囲なので、\(-3 < x < 7\)

考え方2（定義に従って解く）：

\( -5 < x - 2 < 5 \)

各辺に \(2\) を加えます。

\( -5 + 2 < x < 5 + 2 \)

\( -3 < x < 7 \)

例題：|x - 2| > 5

★☆☆

点 \(2\) からの距離が \(5\) より大きい範囲 \(x\) を求めます。

解答

考え方1（距離で考える）：点 \(2\) から距離 \(5\) の点は \(7\) と \(-3\) です。距離が \(5\) より大きいのは、この2点の外側の範囲なので、\(x < -3\) または \(x > 7\)

考え方2（定義に従って解く）：

\( x - 2 < -5 \) または \( x - 2 > 5 \)

それぞれ解きます。

\( x < -5 + 2 \implies x < -3 \)

\( x > 5 + 2 \implies x > 7 \)

よって \(x < -3, x > 7\)

場合分けによる解法

★★★

絶対値を含む方程式や不等式は、絶対値の中身の符号によって場合分けをして解くこともできます。特に、絶対値記号が複数ある場合などに有効です。

基本的な考え方：

  1. 絶対値の中身が \(0\) になる \(x\) の値を求め、それを境界として数直線を区間に分ける。
  2. 各区間ごとに、絶対値の中身の符号を判断し、絶対値記号を外す。
  3. 各区間ごとに、得られた方程式または不等式を解く。
  4. 得られた解が、その区間の条件を満たしているかを確認する。
  5. 各区間で得られた解をすべて合わせる。

例題：不等式 \(|x - 1| + |x - 3| < 4\) を解きなさい。★★★

解答と解説
  1. 絶対値の中身が0になるのは \(x=1\) と \(x=3\) のときです。これにより、数直線を3つの区間 \(x < 1\)、\(1 \le x < 3\)、\(x \ge 3\) に分けます。
  2. 場合1: \(x < 1\) のとき

    \(x-1 < 0\) かつ \(x-3 < 0\) なので、絶対値を外すと両方にマイナスがつきます。

    \( -(x-1) - (x-3) < 4 \)

    \( -x + 1 - x + 3 < 4 \)

    \( -2x + 4 < 4 \)

    \( -2x < 0 \)

    \( x > 0 \)

    条件 \(x < 1\) と合わせて、この場合の解は \(0 < x < 1\)

  3. 場合2: \(1 \le x < 3\) のとき

    \(x-1 \ge 0\) かつ \(x-3 < 0\) なので、\(|x-1|=x-1\)、\(|x-3|=-(x-3)\) となります。

    \( (x-1) - (x-3) < 4 \)

    \( x - 1 - x + 3 < 4 \)

    \( 2 < 4 \)

    これは常に成り立ちます。したがって、条件 \(1 \le x < 3\) の範囲全体が解となります。

  4. 場合3: \(x \ge 3\) のとき

    \(x-1 > 0\) かつ \(x-3 \ge 0\) なので、両方そのまま外します。

    \( (x-1) + (x-3) < 4 \)

    \( 2x - 4 < 4 \)

    \( 2x < 8 \)

    \( x < 4 \)

    条件 \(x \ge 3\) と合わせて、この場合の解は \(3 \le x < 4\)

  5. 解を合わせる: 場合1, 2, 3 で得られた解の範囲 \(0 < x < 1\)、\(1 \le x < 3\)、\(3 \le x < 4\) をすべて合わせます。

    よって、最終的な解は \( 0 < x < 4 \) となります。

解答: \( 0 < x < 4 \)

🧭

不等式の解き方ガイド

★★★

絶対値を含む不等式を解く際の主なアプローチです。

  • 基本形 \(|X| < a\) (a>0) 解釈: 原点からの距離が \(a\) より小さい。 解き方: \( -a < X < a \) として解く。
  • 基本形 \(|X| > a\) (a>0) 解釈: 原点からの距離が \(a\) より大きい。 解き方: \( X < -a \) または \( X > a \) として解く。
  • 応用形 \(|X - p| < a\), \(|X - p| > a\) (a>0) 解釈: 点 \(p\) からの距離で考える。 解き方: 基本形と同様に、\(X\) の部分を \(X-p\) に置き換えて解く。

    \(|X-p|

    \(|X-p|>a \implies X-p < -a \text{ or } X-p > a \implies X < p-a \text{ or } X > p+a\)

  • 場合分け 解釈: 絶対値の中身の符号で場合を分けて考える。 解き方:
    1. 絶対値の中身が0になる値で数直線を区切る。
    2. 各区間で絶対値を外す。
    3. 各区間で不等式を解く。
    4. 解が区間の条件を満たすか確認。
    5. すべての区間の解を合わせる。

    （特に絶対値記号が複数ある場合や、右辺にも変数がある場合に有効）

✏️

練習問題（不等式）

次の不等式を解きなさい。

  1. \( |x| \le 4 \) ★☆☆ ★★☆
    解答

    \(-4 \le x \le 4\)

  2. \( |x| \ge 1 \) ★☆☆ ★★☆
    解答

    \(x \le -1, x \ge 1\)

  3. \( |x + 1| < 2 \) ★☆☆ ★★★
    解答

    \(-2 < x + 1 < 2\)。各辺から1を引くと \(-3 < x < 1\)

  4. \( |x - 3| \ge 4 \) ★★☆ ★★★
    解答

    \(x - 3 \le -4\) または \(x - 3 \ge 4\)。\(x \le -1\) または \(x \ge 7\)

  5. \( |2x + 1| \le 3 \) ★★☆ ★★★
    解答

    \(-3 \le 2x + 1 \le 3\)。各辺から1を引くと \(-4 \le 2x \le 2\)。各辺を2で割ると \(-2 \le x \le 1\)

  6. \( |3x - 2| > 5 \) ★★★ ★★★
    解答

    \(3x - 2 < -5\) または \(3x - 2 > 5\)。
    \(3x < -3 \implies x < -1\)
    \(3x > 7 \implies x > \frac{7}{3}\)
    よって \( x < -1, x > \frac{7}{3} \)

⚠️

間違いやすいポイント

★★★
  • 不等号の向き: \(|X| < a\) と \(|X| > a\) で解の形（内側か外側か）が変わります。不等号の向きを混同しないようにしましょう。「<」は内側、「>」は外側と覚えると良いかもしれません。等号 (\(\le, \ge\)) が含まれる場合も同様です。
  • \(a\) が負の場合の処理: \(|X| < (\text{負の数})\) は解なし、\(|X| > (\text{負の数})\) はすべての実数（または定義域内のすべての実数）となる点に注意が必要です。
  • 場合分けの条件と解の吟味: 場合分けで解いた後、得られた解が場合分けの条件（例: \(x \ge 1\)）を満たしているかを必ず確認（吟味）する必要があります。条件を満たさない解は除外します。
  • 複数の絶対値記号の場合分け: 複数の絶対値記号がある場合、それぞれの絶対値の中身が0になる点で区切り、すべての組み合わせの場合分けを考える必要があります。区間の設定漏れや符号判断ミスに注意しましょう。
  • 最終的な解の結合: 場合分けで得られた解を最後に合わせる際に、範囲を正しく結合（または共通部分を取るなど、問題に応じて）する必要があります。数直線を書いて確認するとミスが減ります。

理解度チェッククイズ2

  1. 不等式 \( |x| \le 2 \) の解はどれですか？
    1. \( x \le -2, x \ge 2 \)
    2. \( -2 \le x \le 2 \)
    3. \( x < -2, x > 2 \)
    解答と解説

    正解は b. \( -2 \le x \le 2 \) です。

    原点からの距離が2以下である範囲を考えます。これは-2と2の間の範囲（両端を含む）になります。

  2. 不等式 \( |x - 1| > 4 \) を解いた結果はどれですか？
    1. \( -3 < x < 5 \)
    2. \( x < -3, x > 5 \)
    3. \( x \le -3, x \ge 5 \)
    解答と解説

    正解は b. \( x < -3, x > 5 \) です。

    点1からの距離が4より大きい範囲を考えます。\(1-4=-3\) と \(1+4=5\) の外側の範囲なので、\(x < -3\) または \(x > 5\) となります。定義に従って解くと \(x-1 < -4\) または \(x-1 > 4\) となり、これを解いても同じ結果が得られます。

授業全体の要約 🔖

★★★

本日の授業では、絶対値の定義に基づいて、絶対値を含む方程式と不等式の解き方を体系的に学びました。

  • 絶対値の基本: 絶対値 \(|a|\) は原点からの距離であり、常に0以上です。外す際は中身の符号で場合分けします (\(a \ge 0\) なら \(a\), \(a < 0\) なら \(-a\))。
  • 絶対値を含む方程式:
    • \(|X| = a\) (a>0) の解は \(X = \pm a\)。距離が \(a\) である点を求めることに相当します。
    • \(|X - p| = a\) (a>0) の解は \(X = p \pm a\)。点 \(p\) からの距離が \(a\) である点を求めることに相当します。
  • 絶対値を含む不等式:
    • \(|X| < a\) (a>0) の解は \(-a < X < a\)。原点からの距離が \(a\) より小さい範囲。
    • \(|X| > a\) (a>0) の解は \(X < -a, X > a\)。原点からの距離が \(a\) より大きい範囲。
    • \(|X - p|\) の場合も、点 \(p\) からの距離として同様に考えます。
  • 場合分け: 絶対値記号の中身が0になる値を境界として区間を分け、各区間で絶対値を外して解き、最後に解を合わせる方法も有効です（特に絶対値記号が複数の場合）。

絶対値の方程式・不等式は、「距離」という基本的な意味を理解することが重要です。パターンを暗記するだけでなく、なぜその解き方になるのかを常に意識するようにしましょう。
