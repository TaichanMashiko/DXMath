はじめに - 授業連絡と復習

本日は、教科書77ページの「2次関数の最大・最小」から学習を始めます。テストが2週間後に迫っているので、集中して取り組みましょう。

まず、前回の内容の復習です。関数 \(y=f(x)\) における定義域とは \(x\) の範囲のことで、値域とは \(y\) の範囲のことでした。この基本的な用語はしっかりと覚えておきましょう。

1. 放物線の特徴 📖 ★★★

2次関数のグラフは放物線と呼ばれます。放物線にはいくつかの重要な特徴があります。

頂点と軸

頂点（ちょうてん）: 放物線の最も尖った部分。グラフが折り返す点です。

軸（じく）: 放物線が対称となる直線のこと。頂点を通り、放物線を左右対称に分けます。

グラフの凸の向き

放物線は、\(x^2\) の係数（一般的に \(a\) と表される）の符号によって、開く向き（凸の向き）が決まります。

  • \(a > 0\) のとき: グラフは下に凸（谷型）。頂点で最小値をとります。
  • \(a < 0\) のとき: グラフは上に凸（山型）。頂点で最大値をとります。

⭐ この凸の向きと頂点の関係は、最大値・最小値を求める上で非常に重要です。

2. グラフの作図と平行移動の理解 ✍️ ★★★

2次関数のグラフを正確に描くためには、まず基本形を理解し、それがどのように平行移動するかを把握する必要があります。

基本形: \(y=ax^2\)

最も基本的な2次関数のグラフは \(y=ax^2\) です。このグラフの頂点は原点 \((0,0)\) であり、軸は \(y\)軸（直線 \(x=0\)）です。

✍️ 例: \(y=2x^2\) のグラフ

このグラフは下に凸で、頂点は \((0,0)\) です。いくつかの点をプロットしてみましょう。

  • \(x=0 \Rightarrow y=0\)
  • \(x=1 \Rightarrow y=2\)
  • \(x=2 \Rightarrow y=8\)
  • \(x=-1 \Rightarrow y=2\)
  • \(x=-2 \Rightarrow y=8\)

y軸方向の平行移動: \(y=ax^2+q\) ★☆☆ ★★☆

\(y=ax^2\) のグラフを \(y\)軸方向に \(q\) だけ平行移動すると、\(y=ax^2+q\) のグラフになります。頂点は \((0, q)\)、軸は \(y\)軸（直線 \(x=0\)）のままです。

✍️ 例: \(y=2x^2+4\) のグラフ

これは \(y=2x^2\) のグラフを \(y\)軸方向に \(+4\) だけ平行移動したものです。頂点は \((0, 4)\) になります。

x軸方向の平行移動: \(y=a(x-p)^2\) ★☆☆ ★★☆

\(y=ax^2\) のグラフを \(x\)軸方向に \(p\) だけ平行移動すると、\(y=a(x-p)^2\) のグラフになります。頂点は \((p, 0)\)、軸は直線 \(x=p\) になります。

⭐ 注意: \(x\)軸方向の移動では、\(x-p\) のように符号が逆になる点に注意してください。\(p\) だけ移動なら \((x-p)\)、\(-p\) だけ移動なら \((x-(-p)) = (x+p)\) となります。

✍️ 例: \(y=2(x-3)^2\) のグラフ

これは \(y=2x^2\) のグラフを \(x\)軸方向に \(+3\) だけ平行移動したものです。頂点は \((3, 0)\) になります。

一般形への拡張: \(y=a(x-p)^2+q\) ★★☆ ★★★

\(y=ax^2\) のグラフを \(x\)軸方向に \(p\)、\(y\)軸方向に \(q\) だけ平行移動すると、\(y=a(x-p)^2+q\) のグラフになります。これが2次関数の標準形です。

2次関数の標準形: \(y = a(x-p)^2 + q\)

  • 頂点: \((p, q)\)
  • 軸: 直線 \(x=p\)

✍️ 例: \(y=2(x-3)^2+4\) のグラフ

これは \(y=2x^2\) のグラフを \(x\)軸方向に \(+3\)、\(y\)軸方向に \(+4\) だけ平行移動したものです。頂点は \((3, 4)\) になります。

理解度チェッククイズ ❓ ★★☆

  1. 関数 \(y = -3(x+1)^2 - 2\) の頂点の座標と軸の方程式を答えなさい。
    1. 頂点(1, -2), 軸 x=1
    2. 頂点(-1, -2), 軸 x=-1
    3. 頂点(1, 2), 軸 x=1
    解答と解説

    解答: b. 頂点(-1, -2), 軸 x=-1

    解説: 標準形 \(y=a(x-p)^2+q\) と比較すると、\(a=-3, p=-1, q=-2\) です。したがって、頂点は \((-1, -2)\)、軸は \(x=-1\) です。

3. 定義域と値域の再確認 📖 ★★☆

グラフを描く際には、定義域（xの範囲）と、それに対応する値域（yの範囲）を意識することが大切です。特に問題で定義域が指定されている場合は、その範囲内でのグラフの動きに注目し、最大値・最小値を考えます。

💡 グラフを描く練習の重要性: グラフを自分で描けるようになると、最大値・最小値の問題が格段に解きやすくなります。面倒くさがらずに、いくつかの点をプロットして、滑らかな放物線を描く練習をしましょう。

4. 例題演習: 固定された定義域における最大値・最小値 ✍️

問9, 例題7 ★★☆ ★★★

問9(1): \(y = 2x^2\) （定義域 \(1 \leq x \leq 2\)）

解答と解説

グラフは下に凸、頂点は \((0,0)\) です。定義域は \(1 \leq x \leq 2\) なので、グラフの右側の一部を考えます。

  • \(x=1\) のとき、\(y=2(1)^2=2\) (最小値)
  • \(x=2\) のとき、\(y=2(2)^2=8\) (最大値)

答え: 最大値 8 (x=2のとき), 最小値 2 (x=1のとき)

例題7: \(y = -2x+7\) （定義域 \(-1 \leq x \leq 3\)）

➕ 注: 授業ではこの問題を問9(2)として扱っていましたが、教科書等では一次関数の例として扱われることが多い内容です。ここでは例題7として提示します。

解答と解説

この関数は一次関数で、傾きが-2（負）なので右下がりの直線です。

  • \(x=-1\) のとき、\(y=-2(-1)+7 = 2+7=9\) (最大値)
  • \(x=3\) のとき、\(y=-2(3)+7 = -6+7=1\) (最小値)

答え: 最大値 9 (x=-1のとき), 最小値 1 (x=3のとき)

⚠️ 間違いやすいポイント ★★☆

  • グラフの平行移動の方向の誤解: \(y=a(x-p)^2+q\) の形において、\(x\)軸方向の移動は \(p\)（符号が逆）、\(y\)軸方向の移動は \(q\)（符号そのまま）です。特に \(x\)軸方向の \(p\) の符号を間違えやすいので注意が必要です。
  • 定義域の端だけを見て判断するミス: 2次関数の最大・最小は、必ずしも定義域の端でとるとは限りません。頂点が定義域内にあれば、頂点が最大値または最小値になる可能性があります。
  • 平方完成の省略: グラフの頂点を正確に求めるためには平方完成が不可欠です。暗算で済ませようとせず、丁寧に計算する習慣をつけましょう。
  • 「最大値なし」「最小値なし」の判断: 定義域が制限されていない場合や、グラフが定義域の端を含まない場合など、「なし」と答えるべき状況を正確に判断することが大切です。

💡 グラフ作成のコツ ★★★

  • 頂点を最優先でプロット: まず平方完成して頂点の座標を求め、グラフ用紙にプロットします。これがグラフの基準点となります。
  • 軸を描く: 頂点を通る縦線（軸 \(x=p\)）を薄く描くと、左右対称性を意識しやすくなります。
  • \(y\)切片を求める: \(x=0\) を代入して \(y\)切片（グラフが\(y\)軸と交わる点）を求めると、グラフの通過点が一つ増え、より正確なグラフが描けます。
  • もう1～2点プロット: 頂点の近くや定義域の端など、あと1～2点計算してプロットすると、グラフの曲がり具合がより正確になります。軸に対称な点を利用するのも有効です。
  • 滑らかな曲線で結ぶ: プロットした点を、放物線の特徴（滑らかな曲線）を意識して結びます。直線でカクカクと結ばないように注意しましょう。
  • 定義域を明示する: 定義域が指定されている場合は、グラフ上でその範囲を太線で示すなど、明確に区別できるようにしましょう。

✏️ 標準的な練習問題

問題 1 ★★☆ ★★☆

関数 \(y = -(x+2)^2 + 1\) のグラフを描き、その頂点と軸、および定義域が \(-3 \leq x \leq 0\) のときの最大値と最小値を求めなさい。

解答と解説

この関数は上に凸の放物線です。

頂点: \((-2, 1)\)

軸: \(x = -2\)

定義域 \(-3 \leq x \leq 0\):

頂点 \(x=-2\) は定義域内にあります。上に凸なので、頂点が最大値です。
最大値: \(1\) (\(x=-2\) のとき)

最小値を求めるために定義域の端の値を計算します。
\(x=-3\) のとき: \(y = -(-3+2)^2 + 1 = -(-1)^2 + 1 = -1 + 1 = 0\)
\(x=0\) のとき: \(y = -(0+2)^2 + 1 = -(2)^2 + 1 = -4 + 1 = -3\)
よって、最小値は \(-3\) (\(x=0\) のとき)。

答え: 頂点 \((-2, 1)\), 軸 \(x=-2\)。
定義域 \(-3 \leq x \leq 0\) において、最大値 1 (x=-2のとき), 最小値 -3 (x=0のとき)

🔖 授業の要点まとめ ★★★

  • 2次関数 \(y=ax^2+bx+c\) のグラフは放物線。
  • 標準形 \(y=a(x-p)^2+q\) から、頂点 \((p,q)\)、軸 \(x=p\) がわかる。
  • \(a>0\) なら下に凸（頂点で最小値）、\(a<0\) なら上に凸（頂点で最大値）。
  • 定義域が指定されている場合、最大値・最小値は「頂点のy座標」と「定義域の端のy座標」を比較して決定する。
  • グラフを描くことで、最大値・最小値の判断が容易になる。定義域の範囲を意識してグラフを正確に描くことが重要。
  • 表を作成して具体的なx,yの値を求めることは、グラフの形状を理解する上で有効。
